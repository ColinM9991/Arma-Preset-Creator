FAQ

Arma Preset Creator was developed in 2016, by one person playing with a small community of friends, called Family of Gamers, and aimed to solve the problem of having multiple preset configurations for Arma 3. With no native support for importing Workshop collections as presets, the Arma Preset Creator integrated with the Steam Web API to seamlessly generate a preset file capable of being imported to the Arma 3 Launcher, while including all dependencies and sub-dependencies. There have been several revisions over the years. Most recent has been a top-level domain change to revive the project after the developer temporarily stepped away in 2021.

The Arma Preset Creator has also inspired many to create their own, similar concepts, as well as receiving positive feedback from many big names in the Arma community.

This tool integrates with the Steam Web API to retrieve all mods in a collection. It traverses the dependency graph as a tree, filtering out any duplicates that it has already retrieved (example: CBA_A3). Doing this means you can download an importable preset file which contains all required dependencies.

There are two ways to use this tool depending on whether you want to download or share a preset.

Downloading a preset

If you have received a link from a fellow community member you simply have to enable or disable any optional mods, of your preference, then click Download.

Otherwise, if you're here to create a preset:

  1. Create or choose the Steam Workshop collections that you will create a preset from.
  2. If you're adding more than one collection, click the green add button next to the Workshop ID input.
  3. Paste the full URL, or just the ID from the URL, in the Workshop ID input(s) above.
  4. Mark any optional collections as optional. This will allow players to toggle on/off certain mods at their preference, such as JSRS.
  5. Type a preset name. While this is optional, it is recommended when using multiple collections. If none is specified, the name of the first mandatory collection is used.
  6. Click 'Create'.
  7. If you have specified an optional collection, enable any mods you'd like to include in the preset.
  8. Scroll to the bottom and click Download.
  9. Import the downloaded preset HTML file in the Arma 3 Launcher > Mods > Preset dropdown.
Sharing a preset
  1. Create or choose the Steam Workshop collections that you will create a preset from.
  2. If you're adding more than one collection, click the green add button next to the Workshop ID input.
  3. Paste the full URL, or just the ID from the URL, in the Workshop ID input(s) above.
  4. Mark any optional collections as optional. This will allow players to toggle on/off certain mods at their preference, such as JSRS.
  5. Type a preset name. While this is optional, it is recommended when using multiple collections. If none is specified, the name of the first mandatory collection is used.
  6. Click 'Share'. This will copy a link to your clipboard which you can share on Discord, or your community forums.
